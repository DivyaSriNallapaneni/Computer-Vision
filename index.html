Module 3 – Object Analysis

Demo Video

This module analyzes object images using five scripts: Segmentation, Boundary Detection, Edge & Corner Detection, Gradients, and Laplacian of Gaussian.

Segmentation – ArUco Markers (aruco_segmentation.py)

Segmentation of ArUco marker images. Each input objN.jpg has a corresponding objN_segmented.jpg output.

Input Images (module3/static/images_aruco)
Segmented Outputs (module3/static/results)

Boundary Detection (boundary_detection.py)

Boundary/edge detection on the original object images.

Input Images (module3/static/images_original)
Boundary / Edge Outputs (module3/static/results)

Edge & Corner Detection (edge_corner.py)

Corner feature detection on each original image.

Input Images (module3/static/images_original)
Corner Outputs (module3/static/results)

Gradients (gradients.py)

Gradient magnitude (and angle) calculated for each original image.

Input Images (module3/static/images_original)
Gradient Magnitude Outputs (module3/static/results)

Laplacian of Gaussian (laplacian.py)

LoG filter highlighting blob-like structures and fine details.

Input Images (module3/static/images_original)
LoG Outputs (module3/static/results)
