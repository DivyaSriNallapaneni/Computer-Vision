Module 1 – Real-World Dimension from a Single Image

Demo Video – Module 1 in Action

This video shows the Module 1 web application estimating the real-world length of an object from a single calibrated image, using the perspective projection relation X = Z · x / f.

Interactive Web Application

Upload an image of the object (e.g., a cup), enter the camera parameters, then click two points on the object in the image (for example, the left and right edges of the cup rim). The app estimates the real-world length based on the distance from camera to object, focal length, and sensor width.

Evaluation – Same Cup at Different Distances

For evaluation, we used a single object (a cup) and measured its true width (e.g., rim-to-rim) using a ruler. We then captured images at 10 different camera distances and used this web app to estimate the length from each image. Each time we measure, we can click "Add this result to evaluation table" to automatically add a row below.

Error is computed as: Error (%) = |Predicted − True| / True × 100.
