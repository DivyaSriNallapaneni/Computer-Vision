Q2(i) – Real-time Object Tracker using Marker

This tracker uses a visual marker (e.g., QR code / AprilTag / ArUco) to localize and track an object in real time. The marker pose is detected in each frame and used as the object’s position.

Q2(ii) – Real-time Markerless Tracker

This version tracks the object without any explicit marker. It can use KLT feature tracking, template matching, or color-based tracking to follow the object across frames.
