Segmentation – ArUco Markers (aruco_segmentation.py)

This script segments the ArUco marker images. Each input objN.jpg has a corresponding objN_segmented.jpg output.

Segmented Outputs (static/results)

Boundary Detection (boundary_detection.py)

This script performs boundary/edge detection on the original object images.

Input Images (static/images_original)
Boundary / Edge Outputs (static/results)

Outputs follow the naming pattern imgN_edges.jpg in static/results/ for N = 1..10.

Edge & Corner Detection (edge_corner.py)

This script focuses on detecting corner features on each original image.

Corner Detection Outputs (static/results)
